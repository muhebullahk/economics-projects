\hline

Cmd + W & Close current tab \\

\section*{Daily Workflow (Recommended)}

\begin{enumerate}
\item Save file (Cmd + S)
\item Check changes: \texttt{git status}
\item Stage changes: \texttt{git add .}
\item Commit changes: \texttt{git commit -m "message"}
\item Sync to GitHub: \texttt{git push}
\end{enumerate}
